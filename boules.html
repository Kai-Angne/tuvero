Optionen und Einstellungen

Import & Export

Spielernamen für Autovervollständigung aus dem Internet laden.

Dieser Browser kann keine lokalen Dateien laden.

Tabellen-Export

Exportierte csv-Dateien lassen sich bequem mit Office-Programmen wie Excel, LibreOffice und Google Docs öffnen.

Damit die Datei geöffnet werden kann, muss sie mit der .csv-Dateiendung gespeichert werden, beispielsweise als 'boules.csv'.


Download
ACHTUNG: Dieser Browser unterstützt wichtige Standards nicht, die zum Speichern und Laden notwendig sind. Beim Schließen des Browserfensters gehen sämtliche Daten verloren.

Turnier speichern

Speichert das aktuelle Turnier in einer Datei. Diese kann später wieder geladen werden, um das Turnier wieder herzustellen.

Um das Turnier nach Schließen des Browsers fortzusetzen, muss der Zustand manuell in einer Datei zwischengespeichert oder ein aktueller Browser installiert werden.



Download

Turnier laden

Das Turnier wird aus einer Datei neu geladen. Eventuell laufende Turniere werden überschrieben.

Dieser Browser kann keine lokalen Dateien laden. Aktualisieren sie ihren Browser oder öffnen sie diese Seite in einem moderneren Browser. Manche Hersteller stellen keine Updates für ihre Browser unter einigen Betriebssystemen zur Verfügung (z.B. Apple mit Safari für Windows).

Storage

Dieses Programm und alle Eingaben werden im Browser gespeichert, um auch offline und nach Beenden des Browsers fortfahren zu können. Dazu werden aktuelle Turnierstände nach jeder Änderung automatisch gespeichert.
