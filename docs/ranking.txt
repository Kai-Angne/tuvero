ranking.js

A class for player (and team) rankings using matrices for storage and
processing purposes.


Elements:
Games (a square counting how often someone played against another player

Netto (a vector counting the netto points)

Wins (a square counting the wins against another player)
  "line won a game against row"
  line sum: number of wins
  row sum: number of losses

Serialization:

[Games (square.toString)]
--\n
[Wins (square.toString)]
--\n
[Netto (line vector)]
