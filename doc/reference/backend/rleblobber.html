RLEBlobber: A blobber for multi-dimensional sparse non-circular arrays, intended for numerical data. Compression occurs using run-length-encoding of all entries that represent null in some form, e.g. undefined, null and 0

Contents of the array can only be numbers at this point

This object can work with encapsulated instances or static functions. Use what suits you best, they're calling each other anyway.

Argument: array

a reference to the array

Returns: {RLEBlobber} this
