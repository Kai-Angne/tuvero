Emitter.prototype.emit = function(event, data)

Emits an event to all registered listeners by calling the callback functions of format 'on'+event, if available

The callback functions' parameters are the emitter (this), the event string (without 'on') and an optional data object, in this order.

Argument: event

a string containing the name of the event. Callback functions need to have a function name in the format 'on'+event Argument: data

arbitrary additional data. Please keep it simple!

Returns: true if the some listener received the event, false otherwise

Emitter.prototype.validEvent = function(event)

Argument: event

Returns: true if the event type is defined, false otherwise
